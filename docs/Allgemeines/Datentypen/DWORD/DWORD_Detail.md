# Datentyp-Detail: DWORD

```{index} single: Datentyp-Detail: DWORD
```

## Beschreibung
Bitfolge der Länge 32

## Normative Definition
Gemäß **DIN EN 61131-3** wird dieser Datentyp wie folgt spezifiziert:

| Eigenschaft | Wert |
| :--- | :--- |
| **Bits** | 32 |
| **Wertebereich** | Bitfolge (32 Bit) |
| **Referenz** | Tabelle 10, Nr. 20 |

## Verwendung in der Programmierung
In der IEC 61131-3 und IEC 61499 wird dieser Typ zur Deklaration von Variablen und in Funktionsbausteinen verwendet.
Bei der Verwendung als Literal kann ein Präfix verwendet werden, z.B. `DWORD#...`.
