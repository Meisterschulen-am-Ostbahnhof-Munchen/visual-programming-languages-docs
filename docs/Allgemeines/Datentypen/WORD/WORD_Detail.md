# Datentyp-Detail: WORD

```{index} single: Datentyp-Detail: WORD
```

## Beschreibung
Bitfolge der Länge 16

## Normative Definition
Gemäß **DIN EN 61131-3** wird dieser Datentyp wie folgt spezifiziert:

| Eigenschaft | Wert |
| :--- | :--- |
| **Bits** | 16 |
| **Wertebereich** | Bitfolge (16#0000 bis 16#FFFF) |
| **Referenz** | Tabelle 10, Nr. 19 |

## Verwendung in der Programmierung
In der IEC 61131-3 und IEC 61499 wird dieser Typ zur Deklaration von Variablen und in Funktionsbausteinen verwendet.
Bei der Verwendung als Literal kann ein Präfix verwendet werden, z.B. `WORD#...`.
