# A2X (BOOL)

```{index} single: A2X (BOOL)
```

unidirectional Adapter Interface for 2 Events and 2 Bools

## Interface

### Events

| Name | Comment | With |
| :--- | :--- | :--- |
| E_UP | UP | UP |
| E_DOWN | DOWN | DOWN |

### Data

| Name | Type | Comment |
| :--- | :--- | :--- |
| UP | BOOL | TRUE = forward, up, right, clockwise |
| DOWN | BOOL | TRUE = backward, down, left, counter-clockwise |
