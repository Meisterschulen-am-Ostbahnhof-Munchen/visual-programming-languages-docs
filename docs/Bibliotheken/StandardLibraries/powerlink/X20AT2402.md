# X20AT2402

```{index} single: X20AT2402
```

* * * * * * * * * *

## Einleitung
Der X20AT2402 Funktionsblock dient zum Einlesen von Temperaturwerten über den openPOWERLINK Feldbus. Er entspricht dem B&R X20AT2402 Temperatur-Eingangsmodul (2 Kanäle).

## Schnittstellenstruktur

### **Ereignis-Eingänge**
- **INIT**: Initialisierung.
- **REQ**: Datenaktualisierung anfordern.

### **Ereignis-Ausgänge**
- **INITO**: Initialisierung bestätigt.
- **CNF**: Neue Temperaturdaten verfügbar.

### **Daten-Eingänge**
- **QI** (BOOL): Qualifizierer.
- **CNID** (USINT): Client Node ID.
- **MODID** (UINT): Modul-ID.

### **Daten-Ausgänge**
- **QO** (BOOL): Qualifizierer.
- **CNIDO** (USINT): Bestätigte Node ID.
- **STATUS** (STRING): Status.
- **T01** (REAL): Temperatureingang 01 (typischerweise in °C, abhängig von Modulkonfiguration).
- **T02** (REAL): Temperatureingang 02.

## Funktionsweise
Liest Temperaturwerte vom physischen Modul und stellt sie als REAL-Werte zur Verfügung.
