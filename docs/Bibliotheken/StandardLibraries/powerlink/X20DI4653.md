# X20DI4653

```{index} single: X20DI4653
```

* * * * * * * * * *

## Einleitung
Der X20DI4653 Funktionsblock dient zum Einlesen von digitalen Signalen über den openPOWERLINK Feldbus. Er entspricht dem B&R X20DI4653 Digital-Eingangsmodul (4 Kanäle).

## Schnittstellenstruktur

### **Ereignis-Eingänge**
- **INIT**: Initialisierung.
- **REQ**: Eingangsdaten aktualisieren.

### **Ereignis-Ausgänge**
- **INITO**: Initialisierung bestätigt.
- **CNF**: Neue Daten verfügbar.

### **Daten-Eingänge**
- **QI** (BOOL): Qualifizierer.
- **CNID** (USINT): Client Node ID.
- **MODID** (UINT): Modul-ID.

### **Daten-Ausgänge**
- **QO** (BOOL): Qualifizierer.
- **CNIDO** (USINT): Bestätigte Node ID.
- **STATUS** (STRING): Status.
- **DI01** (BOOL): Digitaler Eingang 01.
- **DI02** (BOOL): Digitaler Eingang 02.
- **DI03** (BOOL): Digitaler Eingang 03.
- **DI04** (BOOL): Digitaler Eingang 04.

## Funktionsweise
Liest den Zustand der digitalen Eingänge vom Modul und stellt sie als Boolesche Werte bereit.
