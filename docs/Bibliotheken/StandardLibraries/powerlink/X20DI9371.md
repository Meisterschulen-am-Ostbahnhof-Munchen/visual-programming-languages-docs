# X20DI9371

```{index} single: X20DI9371
```

* * * * * * * * * *

## Einleitung
Der X20DI9371 Funktionsblock dient zum Einlesen von digitalen Signalen über den openPOWERLINK Feldbus. Er entspricht dem B&R X20DI9371 Digital-Eingangsmodul (12 Kanäle).

## Schnittstellenstruktur

### **Ereignis-Eingänge**
- **INIT**: Initialisierung.
- **REQ**: Eingangsdaten aktualisieren.

### **Ereignis-Ausgänge**
- **INITO**: Initialisierung bestätigt.
- **CNF**: Neue Daten verfügbar.

### **Daten-Eingänge**
- **QI** (BOOL): Qualifizierer.
- **CNID** (USINT): Client Node ID.
- **MODID** (UINT): Modul-ID.

### **Daten-Ausgänge**
- **QO** (BOOL): Qualifizierer.
- **CNIDO** (USINT): Bestätigte Node ID.
- **STATUS** (STRING): Status.
- **DI01** - **DI12** (BOOL): Digitale Eingänge 01 bis 12.

## Funktionsweise
Liest den Zustand der 12 digitalen Eingänge vom Modul.
