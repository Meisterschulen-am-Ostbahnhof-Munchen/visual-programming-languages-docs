# X20DO4649

```{index} single: X20DO4649
```

* * * * * * * * * *

## Einleitung
Der X20DO4649 Funktionsblock dient zur Ansteuerung von digitalen Ausgängen über den openPOWERLINK Feldbus. Er entspricht dem B&R X20DO4649 Digital-Ausgangsmodul (4 Kanäle).

## Schnittstellenstruktur

### **Ereignis-Eingänge**
- **INIT**: Initialisierung.
- **REQ**: Ausgänge schreiben.

### **Ereignis-Ausgänge**
- **INITO**: Initialisierung bestätigt.
- **CNF**: Schreibvorgang bestätigt.

### **Daten-Eingänge**
- **QI** (BOOL): Qualifizierer.
- **CNID** (USINT): Client Node ID.
- **MODID** (UINT): Modul-ID.
- **DO01** (BOOL): Digitaler Ausgang 01.
- **DO02** (BOOL): Digitaler Ausgang 02.
- **DO03** (BOOL): Digitaler Ausgang 03.
- **DO04** (BOOL): Digitaler Ausgang 04.

### **Daten-Ausgänge**
- **QO** (BOOL): Qualifizierer.
- **CNIDO** (USINT): Bestätigte Node ID.
- **STATUS** (STRING): Status.

## Funktionsweise
Setzt die digitalen Ausgänge am Modul entsprechend den Eingangswerten `DO01`-`DO04`.
