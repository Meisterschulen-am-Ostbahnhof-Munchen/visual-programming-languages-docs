# EMB_RES

```{index} single: EMB_RES
```

* * * * * * * * * *

## Einleitung
Eine grundlegende Embedded-Ressource für die Ausführung von Funktionsblock-Netzwerken. Sie beinhaltet standardmäßig einen E_RESTART Funktionsblock für Start/Stopp-Ereignisse.
