# PKP_2200

* * * * * * * * * *

## Einleitung
PKP_2200 ist eine globale Konstante für die Blink Marine PowerKey-Schnittstelle. Diese Konstante dient als Standardwert für die Konfiguration von Digital-Input-Funktionalitäten im BlinkMarine-System.

## Funktionsweise
PKP_2200 definiert eine globale Konstante vom Typ USINT (Unsigned Short Integer) mit dem hexadezimalen Wert 16#21 (dezimal 33). Diese Konstante wird als Standardwert für die PowerKey-Konfiguration verwendet.

## Technische Besonderheiten
- **Datentyp**: USINT (8-Bit vorzeichenloser Integer)
- **Initialwert**: USINT#16#21 (hexadezimal 21, dezimal 33)
- **Paketzuordnung**: BlinkMarine::io::DI

## Zustandsübersicht
Als globale Konstante besitzt PKP_2200 keine Zustandsübergänge und behält während der gesamten Laufzeit den fest definierten Wert.

## Anwendungsszenarien
- Standardkonfiguration für Blink Marine PowerKey-Interfaces
- Verwendung als Referenzwert in Digital-Input-Funktionsblöcken
- Konfiguration von I/O-Parametern in maritimen Steuerungssystemen

## Vergleich mit ähnlichen Bausteinen
Im Gegensatz zu Funktionsblöcken mit dynamischem Verhalten handelt es sich bei PKP_2200 um eine statische Konstante ohne Verhaltenslogik. Sie dient ausschließlich der Bereitstellung eines festen Konfigurationswerts.

## Fazit
PKP_2200 stellt einen essentiellen Grundbaustein für die Konfiguration von Digital-Input-Schnittstellen im BlinkMarine-System dar. Durch die Verwendung als globale Konstante ermöglicht sie eine zentrale und konsistente Konfiguration über verschiedene Funktionsblöcke hinweg.
