# DualOut ESP32S3
## IO

### Digitale Ausgänge

Diese Version ist erkennbar weil sie einen 6-poligen SKEDD Stecker hat und zusätlich einen 4-poligen SKEDD. 

| Ausgang:  | PIN (ESP32S3) |
|-----------|---------------|
| Output_Q1 | 21            |
| Output_Q2 | 46            |
